## Description

Adds [Base Identity & Reputation](https://github.com/n-burlakov/base-identity-reputation) to the awesome-base list.

**Base Identity & Reputation** is an on-chain identity and reputation system built for the Base ecosystem. It provides:

- **On-chain identity profiles**: Users register their Ethereum address with metadata (IPFS, URL, or JSON)
- **Reputation system**: Registered users can give reputation points to other addresses (up to 100 per pair)
- **Public profile lookup**: Anyone can query registration status, metadata, and reputation scores

## Details

- **Network**: Base Sepolia (chainId 84532)
- **Contract**: Deployed and verified on [BaseScan](https://sepolia.basescan.org/address/0x5B3Ff49Cd951c676F982B14c10eDD0E19056Ac2e)
- **Live Demo**: [https://base-identity-reputation.vercel.app/](https://base-identity-reputation.vercel.app/)
- **Tech Stack**: Solidity + Hardhat (contracts), React + TypeScript + Vite + wagmi (frontend)
- **License**: MIT

## Use Cases

- Proof-of-humanity / identity verification
- Reputation-based applications (credit scoring, DAO governance)
- Composable building block for DeFi and social dApps

The project is fully open-source, documented, and includes integration examples for other developers.
